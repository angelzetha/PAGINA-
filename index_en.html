External style sheet.

An external CSS style sheet can be applied to any number of HTML documents by placing an element in each HTML Document. The rel attribute of the tag must be set to "stylesheet" and the href attribute to the relative or absolute value. path to style sheet. While the use of relative URL paths is generally considered good practice, absolute paths can be used as well. In HTML5, the type attribute can be omitted. It is recommended that the tag be placed in the tag of the HTML file so that the styles are loaded before the elements they design. Otherwise, users will see a flash of unstyled content.

Internal styles..

Tagged CSS within an HTML document works like an external style sheet, except that it lives in the HTML document you design rather than in a separate file, and therefore can only be applied to the document you live in. Note that this element must be inside the element for HTML validation (although it will work in all current browsers if placed in the body).

Styles online.

Use inline styles to style a specific element. Note that this is not optimal. Placing style rules in a <Style> External CSS file or tag is recommended to maintain a distinction between content and presentation. Inline styles override any CSS in an external <Style> tag or style sheet. While this can be useful in some circumstances, this fact often reduces the maintainability of a project. Headers.

HTML provides not just simple paragraph tags, but six separate heading tags to indicate headings of various sizes and thicknesses. Listed as heading 1 to heading 6, heading 1 has the largest and thickest text, heading 6 is the smallest and thinnest, down to paragraph level. This topic details the proper use of these tags.

Text format.

While most HTML tags are used to create elements, HTML also provides text formatting tags to apply text-related styles to portions of text. This topic includes HTML text formatting examples, such as highlighting, bold text, underline, subscript, and strikethrough.
